# React + Express + PostgreSQL Starter (Sin JWT)

## Estructura
- backend/: Express + Node.js
- frontend/: React (create-react-app structure simplified)

## Pasos (desarrollo local)

### Backend
1. Ir a `backend/`
2. Copiar `.env.example` a `.env` y ajustar credenciales.
3. Instalar dependencias:
   ```
   npm install
   ```

   ```
4. Iniciar servidor:
   ```
   npm run dev
   ```
   (o `npm start`)

### Frontend
1. Ir a `frontend/`
2. Instalar dependencias:
   ```
   npm install
   ```
3. Iniciar:
   ```
   npm start
   ```

4. Al iniciar el sistema se crean roles y un usuario master login -> master@master.com - master
